指针反向
这题我想了一个新的思路和解法，就是每次把链表反向，如果最后又回到头结点，那么则证明有环<br>
开始反转之前<br>
A → B → C<br>
     IDL<br>
A ← B → C   A ← B ← C   A ← B ← C   A ← B ← C<br>
     IDL         IDL         IDJ          KDJ<br>
反转之后<br>
A → B ← C<br>
     KDJ<br>
可以看到，每次把后一个节点指向前一个节点，最终会回到头节点，所以判断如果回到头节点，则表示单
链表有环。<br>
